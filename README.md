### Как запустить
1. Скачать `node.js`
2. Склонировать репозиторий (https://github.com/mrSerega/mrserega.github.io)
3. перейти в корень
4. выполнить команду
```bash
npm i
node index.js
```
5. Перейти по адресу `127.0.0.1:3000/index.html`
